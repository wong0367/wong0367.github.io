Things that I know about Web Development

HTML

Adding Image
We can use the <img> element to add imgaes to a website. There are two required attributes in the <img> tag: scr and alt.
Example: <img src="https://picsum.photos/201" alt="">
Result:
Creating Links
We can use the <a> tag to link one page to another. <href> attribute indicates the link's destination.
Example: <a href="google.com">Click Here</a> to search on Google
Result: Click Here to search on Google
Highlighting Text
We can use the <mark> tag to highlight important text.
Example: <mark>This sentense is important.</mark>
Result: This sentense is important.
Adding HTML Table
We can use the <table> tag to add HTML tables on a webpage. An HTML table is composed of one <table> element and one or more <tr>, <th>, and <td> elements.
Example: <table> <tr> <th>Name</th> <th>Age</th> </tr> <tr> <td>Tony</td> <td>20</td> <tr> </table>
Result:
Name Age
Tony 20
Abbreviation
We can use the <abbr> tag to add defination for abbreviation or an acronym
Example: <abbr title="United States of America">USA</abbr>
Result: USA
